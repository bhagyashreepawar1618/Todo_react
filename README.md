# React Todo App

A simple Todo application built using React and Context API with Tailwind CSS for styling.

---

## Features

- Add new todos
- Edit existing todos
- Mark todos as completed
- Delete todos
- Global state management using Context API

---

## Tech Stack

- React
- Context API
- Tailwind CSS
- JavaScript (ES6)

---

## Project Structure

src  
├── components  
│   ├── TodoForm.jsx  
│   └── TodoItem.jsx  
├── contexts  
│   └── TodoContext.jsx  
├── App.jsx  
├── main.jsx  

---

## Run Locally

1. Clone the repository

git clone https://github.com/bhagyashreepawar1618/Todo_react.git

2. Install dependencies

npm install

3. Start the development server

npm run dev
